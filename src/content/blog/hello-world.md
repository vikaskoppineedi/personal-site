Welcome to my corner of the internet. This is where I'll write about the things
I build — homelab infrastructure, automation, music tooling, and whatever else
I'm tinkering with.

## Why this site exists

I wanted one place to point people to: my music, my projects, and my writing.
It's a static site built with [Astro](https://astro.build), hosted on GitHub
Pages, with the domain wired up entirely as code.
